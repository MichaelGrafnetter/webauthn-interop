# Disconnect-Okta

## SYNOPSIS
Revokes Okta access token.

## SYNTAX

```
Disconnect-Okta [<CommonParameters>]
```

## DESCRIPTION
Revokes the Okta access token cached from the call to `Connect-Okta`.

## EXAMPLES

### EXAMPLE 1
```
Disconnect-Okta
```
